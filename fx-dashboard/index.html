모델 해석

가격 모멘텀과 평균회귀, 거시 팩터를 조합해 기대경로를 계산합니다.

주의사항

현재 국제정세 값은 사용자가 직접 입력하는 시나리오형 변수입니다. 실제 뉴스 데이터 자동화는 별도 API 연동 단계가 필요합니다.

모델 수식

기대수익률은 모멘텀, 평균회귀, 거시 팩터를 합친 로그드리프트로 계산하고, 변동성은 EWMA 기반으로 추정합니다.
